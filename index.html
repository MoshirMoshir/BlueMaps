BlueMaps is an interactive map of the Blue Hole
*Currently in development*

    Features Include:
  • Modular World Map
    •  Ability to select each section of the map!
  • POI Overlays
    •  Find the locations of important points of interest!
     (Only Escape Pods right now, more overlays to come!)
  • Smooth UI and Map Integration
    •  Uses leaflet for seamless map integration on a webpage.
  • Works on Mobile
    •  Designed to work on mobile devices as well!

If anyone would like to help, feel free to send a message in the BlueMaps discussion page on Dave the Diver's Official Discord.
